MySQL安装及基本操作

发表于 2016-09-10   |   分类于 日志

  • 安装

在Ubuntu上安装MySQL只需要三条命令即可，如下：

$sudo apt-get install mysql-server
$sudo apt-get install mysql-client
$sudo apt-get install libmysqlclient-dev

安装过程中需要设置密码，按步骤设置即可。

  • 检查是否安装成功

sudo netstat -tap|grep mysql

上述命令可以检查MySQL是否正确安装。如果看到MySQL的socket处于listen状态则表示安装成功。

  • 基本操作

mysql -u root -p #连接本地数据库
-u 表示登陆的用户名，-p表示登陆使用的密码，输入密码后就可以登陆到MySQL。
show databases; #显示当前的数据库
use mysql; #使用mysql数据库
show tables; #显示数据库中有的表
show columns from table_name from database_name; #显示表中的字段，可以省略 from database_name
show full columns from table_name from database_name; #显示详细字段内容
set names utf8; #设置字符集，方便显示中文
create database lesson; #创建数据库
create table study(id int(11) unsigned NOT NULL AUTO_INCREMENT COMMENT ‘学生id号’,username varchar(30) NOT NULL DEFAULT ‘’ COMMENT ‘学生名字’,class tinyint(3) unsigned NOT NULL,sex enum(‘男’,’女’,’保密’) CHARACTER SET utf8 COLLATE utf8_general_ci NOT NULL DEFAULT ‘保密’ COMMENT ‘性别’,addtime int(10) NOT NULL DEFAULT ‘0’,PRIMARY KEY (id))ENGINE=InnoDB COMMENT = ‘学生表’; #创建表，同时声明各个字段的属性
rename table study to study_name; #修改表的名称，下同
alter table rename to study_name;
insert into study(username,class,sex) values(‘小王’,1,’男’),(‘小四’,1,’女’); #插入一条项到表study中
select username,sex from table_name; #从table_name选择字段显示，或者是 * 代表显示该表所有字段的内容（查询数据）
deleter from table_name where id=1; #删除table_name中id=1的项
drop table table_name; #删除表 table_name
drop database database_name; #删除数据库

如果要远程连接一个数据库，操作如下：

1.在远程服务端设置：

  1. 修改sql的my.cnf文件：

    $sudo gedit /etc/mysql/my.cnf

  2. 注释bind-address,屏蔽只对本地监听

    #bind-address=127.0.0.1

  3. 重启mysql服务，使配置生效：

    $sudo restart mysql

  4. 在远程主机上打开数据库，并使用mysql数据库,将其中一条修改:

    use mysql;
    UPDATE user SET host=’%’ WHERE user=’root’ and host=’127.0.0.1’ ;

2.在客户主机上连接：

  1. 远程连接

    $mysql -h192.168.1.11 -u root -p 123456
    $其中192.168.1.11代表远程主机的IP 123456 代表数据库的密码
